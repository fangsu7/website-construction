关于我们网络空间安全协会

网络空间安全社团是在计算机与计算科学学院业务指导单位支持下的学生自主发起的非营利性社团组织，以维护网络空间安全为基点，开展网络空间安全竞赛活动、网络空间安全技能培训等活动，以促进学生对网络空间安全的学习和践行，涵养核心价值观，培养web安全、逆向工程、应用密码学、二进制安全的理想人才，引导学生学习维护网络空间安全的重要性，学习维护网络空间安全的重要技能，为国家网络安全贡献出自己的一份力，使其成为建设伟大祖国的栋梁之才。

入门介绍

“Web方向所需要学习的内容偏多，涉及到各类编程语言与开发框架，漏洞的类型也偏多，但是不用担心，web方面的安全研究并不会过于深入开发的底层原理，也并不会涉及太多开发的高级应用。web安全比较有趣味性，基本思想是绕过检测并执行命令，比较符合传统黑客的工作。”

Web

web入门资料(点击获取)

“Misc（杂项）是指在计算机中不属于特定领域或主题的内容。它包括了各种各样的工具、库、模块、函数、小贴士等。Misc可以帮助开发者提高生产力，简化编码过程，减少出错率。常见的Misc内容包括常用的Python库、命令行工具、调试技巧、代码风格规范等。学习Misc内容可以让你更加高效地编写代码，并且拓宽你的技术视野。”

Misc

misc入门资料(点击获取)

“Reverse（逆向工程）要求参与者破解和分析已编译的程序代码，以获取隐藏的答案或漏洞。通过理解和逆向工程二进制文件，包括可执行文件、固件或反汇编代码，选手需要解决密码学、加密算法、漏洞利用等问题。CTF Reverse活动旨在锻炼参与者的逆向分析和解决问题的能力，提高对计算机系统运行机制的理解。”

Reverse

reverse入门资料(点击获取)

“Crypto（密码学）要求参与者应用密码学知识解决各种加密挑战。参赛者需要破解密码、分析加密算法、解密隐藏信息，以获取比赛中的“Flag”（标记）。这种竞赛旨在培养密码学技能，包括对对称加密、非对称加密、数字签名和哈希函数等的理解和应用。CTF Crypto挑战能够提升参与者的密码学知识和实践能力，适合想要深入了解加密技术的人群。”

Crypto

crypto入门资料(点击获取)

“Pwn（漏洞利用）要求参与者发现和利用给定程序的漏洞，以获取比赛中的“Flag”（标记）。参赛者需要在理解二进制代码基础上，针对内存溢出、格式化字符串、堆栈溢出等漏洞类型实现攻击或提供有效的解决方案。这种竞赛旨在培养参赛者的漏洞分析和解决问题的能力，提高对二进制程序安全的认识和掌握。CTF Pwn挑战能够提升参与者的技术水平和实践能力，适合想要深入了解二进制程序漏洞的人群。”
